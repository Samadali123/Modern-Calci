# Modern Calci

A clean, modern calculator built with vanilla HTML, CSS, and JavaScript. No libraries, no dependencies — just straightforward DOM manipulation and expression evaluation. Deployed on GitHub Pages.

Live: [samadali123.github.io/Modern-Calci](https://samadali123.github.io/Modern-Calci)


## Features

- Basic arithmetic — addition, subtraction, multiplication, division
- Chained calculations without clearing between operations
- Clear and delete (backspace) functionality
- Keyboard support
- Modern UI with a clean button layout and visual feedback on press
- Fully responsive — works on mobile and desktop



## Project Structure


Modern-Calci/
├── index.html    # Calculator markup and button layout
├── style.css     # Layout, colors, button styles, hover and active states
└── script.js     # Input handling, expression building, and evaluation logic


## Getting Started

No setup needed. Clone and open directly in a browser.

bash
git clone https://github.com/Samadali123/Modern-Calci.git
cd Modern-Calci
open index.html
